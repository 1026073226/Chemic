⚗️ 化学反应操作指南

普通反应示例（制备水）

  1. 点击“下回”
  2. 点击底部手牌中的H（重复两次）
  3. 在中央出现H₂时，点击制取
  4. 点击底部手牌中的O（重复两次）
  5. 在中央出现O₂时，点击制取
  6. 点击🧫，查看刚刚制取的物质（假设现在已制取数个H₂和一个O₂
  7. 点击H₂，弹出菜单
  8. 点击回收
  9. 得到元素力
  10. 若干回合后，假设你的元素力已足以反应
  11. 点击🧫
  12. 点击H₂，点击反应
  13. (以此类推)
  14. 点击💥
  15. 检查方程式是否正确
  16. 点击=
  17. 右边出现2H₂O，弹窗提示：[红色放热/蓝色吸热]“反应熵（实际造成熵）”
  18. 点击2H₂O
  19. 点击🧫，可以看到H₂O已经在里面了
  20. 点击📖，可以看见熵已经变化

核反应特殊流程

  1. 获取H-2、H-3氢同位素
  2. (注：每个H每轮只能轰击一次)
  3. 核聚变反应(搜索“n”可看到方程式)获得n（中子）
  4. U和n反应触发裂变

    💡Tips:


    1. 制作化合物时，若顺序有误，则无法正常进行反应,如：CCaO3(实际上正确顺序应为CaCO3)
    2. 不要同时打开多个页面，可能有Bug！
    3. 建议放入元素数量最好不要超过20个，否则……
    4. 搜索以“元素a+元素b……+元素c*n”的形式进行，如：搜索氢气与氧气参与的反应可搜索“H*2+O*2”
    5. 点击搜索结果方程式的等号可以自动配平
    6. 快速开关元素菜单会产生Bug，再次尝试快速开关有概率修复Bug
